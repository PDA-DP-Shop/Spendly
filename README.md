## <img src="https://media.giphy.com/media/WUlplcMpOCEmTGBtBW/giphy.gif" width="30px"> Features

<div align="center">

### 💰 Core Money Features

|Feature                 |Description                                    |
|:----------------------:|:----------------------------------------------|
|✏️ **Manual Entry**      |Custom numpad, fast and easy to use            |
|📦 **Barcode Scan**      |Scan any product barcode — auto fills details  |
|🧾 **Bill OCR Scan**     |Photo your receipt — app reads it automatically|
|💳 **Payment Tracker**   |Cash, Card, UPI, Bank Transfer                 |
|🔁 **Recurring Expenses**|Set daily, weekly, monthly repeats             |
|💰 **EMI Tracker**       |Track all loans and EMIs in one place          |
|👥 **Split Bill**        |Split with friends, track who owes you         |

### 📊 Analytics & Reports

|Feature                 |Description                           |
|:----------------------:|:-------------------------------------|
|🔥 **Heatmap Calendar**  |GitHub-style spending heatmap         |
|📊 **Smart Charts**      |Donut, Bar, Line charts               |
|📅 **Year vs Year**      |Compare this year with last year      |
|📆 **Weekend vs Weekday**|See when you spend more               |
|🏆 **Spending Score**    |Your monthly financial health score   |
|📄 **Auto PDF Report**   |Monthly report generated automatically|
|🖼️ **Share as Image**    |Beautiful shareable report card       |

### 🎯 Goals & Challenges

|Feature              |Description                      |
|:-------------------:|:--------------------------------|
|🎯 **Savings Goals**  |iPhone, Trip, Emergency Fund etc |
|🚫 **No Spend Days**  |Challenge yourself to spend less |
|✈️ **Trip Planner**   |Separate budget for every trip   |
|🪔 **Festival Budget**|Diwali, Wedding, Birthday planner|

### 🔒 Privacy & Security

|Feature                      |Description                      |
|:---------------------------:|:--------------------------------|
|🔐 **AES-256-GCM**            |Military grade encryption        |
|🔑 **PIN / Pattern / Face ID**|Multiple lock options            |
|👻 **Fake Balance Mode**      |Show fake data to nosy people    |
|🎭 **Decoy PIN**              |Two PINs — real data vs fake data|
|📤 **Encrypted Export**       |Secure transfer between devices  |
|🗑️ **Secure Delete**          |Data overwritten before deletion |

### 🇮🇳 India Specific

|Feature             |Description                       |
|:------------------:|:---------------------------------|
|📱 **UPI QR Scanner**|Scan UPI QR to auto fill merchant |
|🧮 **GST Calculator**|With CGST/SGST breakdown          |
|💬 **SMS Parser**    |Paste bank SMS to auto add expense|
|🌐 **Multi Language**|English, Hindi, Gujarati          |

## <img src="https://media.giphy.com/media/cj87CxfRtrUifF3Ryk/giphy.gif" width="25px"> Tech Stack

<div align="center">

```
┌──────────────────────────────────────────────────────────┐
│                    SPENDLY TECH STACK                    │
├─────────────────┬────────────────────────────────────────┤
│  Frontend       │  React 18 + Vite 5                     │
│  Styling        │  Tailwind CSS (mobile first)           │
│  Animations     │  Framer Motion                         │
│  State          │  Zustand                               │
│  Storage        │  Dexie.js (IndexedDB)                  │
│  PWA            │  Vite PWA Plugin + Service Worker      │
│  Charts         │  Recharts                              │
│  Barcode        │  ZXing.js                              │
│  OCR            │  Tesseract.js                          │
│  Encryption     │  Web Crypto API (AES-256-GCM)          │
│  Biometrics     │  Web Authentication API (WebAuthn)     │
│  Voice          │  Web Speech API                        │
│  PDF            │  jsPDF                                 │
│  Share Image    │  html2canvas                           │
│  Icons          │  Lucide React                          │
│  Fonts          │  Sora + DM Sans (Google Fonts)         │
```

## <img src="https://media.giphy.com/media/QssGEmpkyEOhBCb7e1/giphy.gif" width="25px"> Run Locally

```bash
# Clone the repo
git clone https://github.com/PDA-DP-Shop/spendly.git

# Go into folder
cd spendly

# Install packages
npm install

# Start dev server
npm run dev

# Open on your phone
# http://[your-ip]:5173
```

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## <img src="https://media.giphy.com/media/ZVik7pIojeM0Y/giphy.gif" width="25px"> Project Structure

```
spendly/
│
├── 📁 public/
│   ├── manifest.json          # PWA config
│   ├── service-worker.js      # Offline support
│   ├── _headers               # Security headers
│   └── _redirects             # Cloudflare routing
│
├── 📁 src/
│   ├── 📁 screens/            # All app screens
│   ├── 📁 components/         # Reusable UI pieces
│   ├── 📁 store/              # App state (Zustand)
│   ├── 📁 hooks/              # Custom React hooks
│   ├── 📁 services/           # Database + APIs
│   ├── 📁 utils/              # Helper functions
│   └── 📁 constants/          # Colors, categories etc
│
├── vite.config.js             # Vite + PWA setup
├── tailwind.config.js         # Tailwind config
└── package.json               # Dependencies
```
