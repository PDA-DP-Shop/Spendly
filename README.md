# Spendly Monorepo

This repository contains the Spendly ecosystem.

## Structure

- `apps/spendly-user`: The consumer-facing mobile/web application.
- `apps/spendly-shop`: The vendor-facing shop application.
- `packages/shared`: Shared logic, utilities, and constants.

## Development

Run user app:
```bash
npm run dev:user
```

Run shop app:
```bash
npm run dev:shop
```
