## ✨ Overview

Spendly is a dual-app ecosystem reimagining how businesses and customers interact with a **"Zero-Knowledge"** philosophy. Now featuring **Hardware-Backed Biometric Security** (WebAuthn) and a sleek **Flat Premium** monochrome aesthetic, Spendly ensures your financial data is as beautiful as it is private.

### 🏪 Spendly Shop (Merchant App)
*A high-speed, secure billing powerhouse for the modern business owner.*
- 🛡️ **Biometric Vault**: Protect shop sales and customer books with **Face ID / Fingerprint** hardware binding.
- ⚡ **30-Second Billing**: Create complex invoices with GST and discounts in seconds.
- 📡 **Beam Protocol**: Instantly send bills via **Web NFC**, Dynamic QR, or 6-digit Bill Codes.
- 🔳 **Flat Premium UI**: A distraction-free, black-and-white professional interface.

### 👤 Spendly User (Customer App)
*The ultimate private expense manager with hardware-level privacy.*
- 🔑 **Hardware Security**: Lock your expenses behind device-level biometrics with PIN fallback.
- 🛡️ **Zero-Knowledge Sync**: Add expenses instantly by tapping your phone at the shop.
- 💼 **Smart Cash & Wallets**: Manage physical cash notes and bank balances with AI-driven analytics.
- 📈 **Flat Designer UI**: A minimalist, shadow-free "Aura" design for a high-performance experience.

## 🛠️ Performance & Security

| Security Layer | Protocol | Experience |
| :--- | :--- | :--- |
| **Biometrics** | **WebAuthn / Passkeys** | Face ID & Fingerprint |
| **Encryption** | **AES-256-GCM** | Military-grade hardware encryption |
| **Hashing** | **PBKDF2 (600k iterations)** | High-entropy PIN security |
| **Privacy** | **Local-Only Storage** | 100% Offline, No cloud dependency |

## 🌐 Project Philosophy: "Flat Premium"

We have transitioned away from traditional drop-shadows and vibrant brand colors to a **Flat Premium** design language. 
- **Monochrome Mastery**: Deep blacks and crisp whites for maximum contrast.
- **Shadow-Free**: Using borders and typography for depth instead of Gaussian blurs.
- **Aura Animations**: Subtle, high-frequency micro-interactions (Framer Motion).

## 🔧 Installation & Local Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/PDA-DP-Shop/Spendly.git
   cd Spendly
   ```

2. **Setup Shop App**
   ```bash
   cd apps/spendly-shop && npm install && npm run dev
   ```

3. **Setup User App**
   ```bash
   cd apps/spendly-user && npm install && npm run dev
   ```
